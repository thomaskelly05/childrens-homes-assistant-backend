# IndiCare OS Frontend

## Architecture

This frontend is the modern operational workspace migration for IndiCare OS.

### Stack

- Next.js
- TypeScript
- Tailwind CSS
- Component-first operational architecture

### Reused Operational Systems

The frontend reuses the existing:

- chronology intelligence
- safeguarding orchestration
- operational continuity systems
- assistant orchestration
- review workflows
- FastAPI operational APIs

### Operational Workspace Areas

- chronology workspace
- safeguarding oversight
- operational notifications
- continuity monitoring
- assistant orchestration
- review queues
- operational alerts
- activity feeds

### Development

```bash
npm install
npm run dev
```

### Backend

The frontend automatically rewrites:

```text
/api/*
```

to:

```text
http://localhost:8000/api/*
```

### Status

The legacy runtime frontend is being progressively migrated into reusable component-driven operational architecture.
