# LifeEcho Web Experience

LifeEcho is the emotional continuity and virtual memory box experience inside IndiCare.

Target route:

https://app.indicare.co.uk/life_echo

## Planned frontend stack

- Next.js
- TailwindCSS
- Framer Motion
- React Three Fiber
- WebSockets

## Core experiences

- Emotional timeline
- Playback journeys
- Child memory portal
- Relationship constellations
- Voice memories
- Emotional atmosphere rendering
- Reflective journalling
- Virtual memory vault
