# Children's Homes Assistant Backend

Backend for a residential childcare platform covering:

- authentication and MFA
- young people records
- incidents, risk, plans, chronology, health, education, family contact
- AI-assisted notes and chat
- compliance, reports, handovers, and rostering

## Requirements

- Python 3.9+
- PostgreSQL
- A valid `.env` file
