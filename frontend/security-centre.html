Security guidance

Use these controls when working with sensitive safeguarding and care information.

Use passkeys first
Face ID, Touch ID, Windows Hello, fingerprint, or hardware keys reduce password and phishing risk.
Lock shared devices
Never leave children’s records visible on screen. Revoke sessions from shared or lost devices immediately.
Step up before sensitive actions
Exports, deletions, security changes, and safeguarding actions should require fresh verification.
