# ORB Simplified Product Model

ORB Residential is organised as seven visible rooms plus account settings.

## Main rooms

1. **Chat** — ask ORB anything; converged workflows start here via starters
2. **Dictate** — capture rough notes; suggested outputs include handover, manager summary, action plan
3. **Voice** — hands-free copilot
4. **ORB Write** — professional record/document studio with practice workflow assistant actions

## Library

5. **Templates** — recording/framework library with common workflows highlighted
6. **Documents & Guidance** — guidance, policies, document analyser and knowledge library (merged)
7. **Saved Outputs** — archive of created work

## Account

Profile, Settings, Billing — unchanged.

## Why this improves clarity

Staff see one product with clear rooms instead of many overlapping tools. Power is preserved through starters, templates, lenses and assistant actions rather than duplicate sidebar destinations.
