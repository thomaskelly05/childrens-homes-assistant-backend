# ORB Therapeutic Template Framework

## Canonical source

- **Primary:** `assistant/knowledge/orb_recording_framework.json`
- **Frontend mirror:** `frontend-next/lib/orb/recording/orb-recording-framework.json`
- **Therapeutic supplement:** `frontend-next/lib/orb/recording/orb-therapeutic-writing.ts` (merged at load)

## Person-centred / therapeutic rules

Templates support recording that is:

- Child-centred and trauma-informed
- Non-blaming and emotionally literate
- Factual and observable before interpretation
- Respectful and inspection-ready
- Grounded in adult professional judgement

## Per-template fields (supplement)

- Writing guidance
- Therapeutic, person-centred, child voice prompts
- Trauma-informed and neutral factual prompts
- What to avoid
- Quality, safeguarding, manager oversight checks
- Spelling/grammar reminder

## Core templates updated

General Dictation, Daily Record, Incident Report, Missing From Home, Safeguarding Concern, Physical Intervention, Key Work Session, Manager Summary, Chronology Entry, Handover, Action Plan, Reg 44 Evidence Summary, Reg 45 Reflection, Statement of Purpose Review, Policy Change Summary.
