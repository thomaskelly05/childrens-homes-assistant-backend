# Frontend/backend capability map

This map captures the integration surface audited for the platform pass. Status values:
`wired`, `partially wired`, `not exposed`, `backend missing`, `frontend missing`.

| Capability | Backend route | Backend service/module | Data returned | Frontend area/page/component | Adapter function | Status | Notes | Risks | Next recommended work |
