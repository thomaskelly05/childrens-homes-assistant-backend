# ORB interaction model

ORB is voice-first and never requires text input for core use. It supports voice, captions, transcript mode, typing fallback, touch, large tap targets and keyboard shortcuts.

Embedded ORB stays docked bottom-right, expands contextually, opens immersive mode on request and never blocks workflow unless expanded.

Standalone ORB is full-screen, immersive, ambient, minimal and accessibility-led.

Emotional safety can shorten replies, slow pacing, reduce motion, soften visuals and offer step-by-step support without diagnosing or replacing safeguarding/professional action.
