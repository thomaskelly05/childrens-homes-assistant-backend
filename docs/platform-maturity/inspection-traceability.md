# Inspection traceability

Inspection readiness is traceable evidence infrastructure, not a scorecard.

Canonical DTO:

- `schemas.operational_state.InspectionEvidenceTrace`

Inspection panels should show:

- evidence status
- linked records
- linked chronology
- linked safeguarding
- linked documents
- linked operational states
- review history
- sign-off history
- stale evidence
- missing evidence
- management oversight requirements

## Framework scope

Traceability should cover:

- Reg 44
- Reg 45
- Annex A
- SCCIF
- Quality Standards
- safeguarding evidence
- workforce evidence
- leadership evidence

## Rule

Inspection logic should not be hardcoded per page. Normalize evidence into trace DTOs and render them through shared lifecycle/evidence patterns.
