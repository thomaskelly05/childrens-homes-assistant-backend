# ORB Residential — Product Definition

## Product identity

# ORB Residential

Powered by IndiCare Intelligence.

The AI companion for adults working in children’s residential care.

---

# Product category

ORB Residential is not positioned as:

- a generic chatbot
- a prompt wrapper
- a compliance robot
- a documentation generator

ORB Residential is positioned as:

# Residential care intelligence.

The platform supports:

- safer recording
- safeguarding thinking
- therapeutic reflection
- child-centred language
- operational support
- inspection readiness
- emotional support for staff
- structured shift thinking
- reflective supervision preparation

# Free access policy

Free access should be limited.

Recommended:

- limited trial
- limited demo prompts
- marketing/demo environment

Do NOT provide unlimited free ORB access.

Reason:

- AI cost exposure
- reduced perceived value
- lower conversion
- lower product trust

# Standalone ORB boundaries

ORB Residential must never:

- imply live record access
- imply chronology access
- imply provider oversight visibility
- fabricate evidence
- fabricate safeguarding state
- fabricate actions

If evidence is missing:

ORB must clearly state the limitation.
