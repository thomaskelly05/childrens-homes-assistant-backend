# ORB presence memory

Presence memory stores safe, minimal preference data only: response length, captions, voice/caption mode, reduced motion, high contrast, pacing, interaction style, recent unresolved topic, recent active workflow and last used mode.

Embedded memory is user/home scoped and active-child scoped when operational. Standalone memory is user scoped and standalone only.

ORB does not store raw audio, unnecessary full transcripts, child data in standalone memory or sensitive safeguarding content unless explicitly allowed and scoped.
