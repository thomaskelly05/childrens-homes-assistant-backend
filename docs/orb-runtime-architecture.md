# ORB runtime architecture

ORB shares one safe intelligence foundation but exposes two product experiences.

Embedded ORB lives inside IndiCare OS. It uses RBAC, active-child/home/provider scope, existing assistant orchestration, chronology/evidence retrieval, safeguarding intelligence, document/reporting intelligence and draft-before-write rules.

Standalone ORB is isolated. It has no live OS record access, no active child, no chronology retrieval, no child documents, no home/staff records and no OS memory.

Shared layers include identity, role contract, visual renderer, voice orchestration, accessibility, emotional safety, environment modes, safe failure copy and static regulatory/sector knowledge.
